Projeto autoral em evolução

Souzas Dev

Portfólio, API e plataforma administrativa.

Um projeto criado para transformar estudos em uma aplicação web real, conectando interface, servidor, autenticação e banco de dados.

Frontend no GitHub ↗ Backend no GitHub ↗
01

Visão geral

A Souzas Dev reúne uma Home pública, formulário de contato, tela de autenticação e painel administrativo em uma única solução.

02

Desafio

Desenvolver uma aplicação completa sem frameworks no frontend, mantendo o código organizado e conectando diferentes partes do sistema.

03

Solução

Um site responsivo integrado a uma API em Node.js, com autenticação administrativa e armazenamento das mensagens em SQLite.

O que foi desenvolvido

Funcionalidades aplicadas no projeto.

Experiência pública

  • Site institucional responsivo
  • Apresentação de serviços e projetos
  • Formulário conectado à API
  • Validação dos campos do formulário
  • Indicador de disponibilidade do sistema

Área administrativa

  • Login administrativo protegido
  • Autenticação utilizando JWT
  • Cookie de autenticação HttpOnly
  • Painel de mensagens recebidas
  • Status de mensagens novas, lidas e arquivadas

Estrutura técnica

  • API desenvolvida com Node.js
  • Banco de dados SQLite
  • Frontend e backend separados
  • Organização do código em módulos
  • Controle de versões com Git e GitHub

Aprendizado

Conhecimento aplicado em um projeto real.

Este projeto permitiu praticar a integração entre interface, servidor e banco de dados, além de aprofundar conhecimentos sobre autenticação, APIs e organização de aplicações web.

A Souzas Dev continua em evolução. Novas funcionalidades, melhorias de acessibilidade, testes e a publicação em produção fazem parte dos próximos passos.
